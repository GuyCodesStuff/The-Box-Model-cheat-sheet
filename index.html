CSS Box Model Cheat Sheet

Hello all!

This will be my Cheat Sheet for the topic of Box Models in CSS

Topics to be discussed:

  • Margin
  • Border
  • Padding
  • Content

The Box Model is made up of the 4 properties that I have listed above. They all work together to make up the elements on our websites.

This is an example of a box with margin, border, padding, and content.

Margin

We will start with a short description of the margin.

The margin is the space between the border and the outside of the element. It is the outermost layer of the box model.

It is important to understand

There are many different properties that can be used to style the margin. Some of these properties include:

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left

Border

Next, we will discuss the border.

The border is the layer that goes around the padding and content. It is the second layer of the box model.

It is important to understand

There are many different properties that can be used to style the border. Some of these properties include:

  • border-width
  • border-style
  • border-color

Padding

Next, we will discuss the padding.

The padding is the space between the content and the border. It is the third layer of the box model.

It is important to understand

There are many different properties that can be used to style the padding. Some of these properties include:

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left

Content

Finally, we will discuss the content.

The content is the innermost layer of the box model. It is where the text and images are displayed.

It is important to understand

There are many different properties that can be used to style the content. Some of these properties include:

  • color
  • font-size
  • text-align

Here I will present a cheat sheet to cover some of the elements used in CSS for the Box Model

Property Description Sample Code

  • margin: 20px
  • margin-top: 20px
  • margin-right: 20px
  • margin-bottom: 20px
  • margin-left: 20px

  • padding: 20px
